# Support

KAOS is community-supported open source software.

## Questions

Use GitHub Discussions when available, or open a normal issue with:

- KAOS version or commit
- install method
- OS and Python version
- command you ran
- expected behavior
- actual behavior

## Bugs

Use the bug report template and include logs with secrets removed.

Do not paste:

- API keys or provider tokens
- Telegram session data or user IDs
- `.env` files
- private hostnames or server IPs
- live workspace notes or memory files

## Security

Do not open public security issues. Follow `SECURITY.md`.
