# Heartbeat — Nexus

## Periodic Checks

### Business Metrics
- Check key project metrics (traffic, conversions, revenue) for anomalies > 15%
- Compare with previous period and identify trends
- Flag metrics that are "too good" — verify methodology

### Competitive Intelligence
- Monitor competitor activity (new launches, pricing changes, feature updates)
- Assess: tactical move or strategic inflection point (10X change)?
- Cross-reference with market trends

### SEO & Organic
- Monitor organic traffic trends across projects
- Check for ranking drops or gains on key queries
- Flag algorithm update signals if detected

### Project Health
- Review active Notion tasks: overdue items, blocked tasks
- Check if any project metrics contradict stated goals
- Identify stagnation patterns — no news != all good

## Red Flags (auto-alert)
- Any metric changed > 15% without explanation
- Competitor did something for the first time
- Two+ unrelated signals pointing in the same direction
- Extended stagnation without changes
